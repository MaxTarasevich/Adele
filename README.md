<br>
Использовалась сборка gulp https://habr.com/ru/post/484714/
<br>
Сайт просмотреть можно здесь https://maxtarasevich.github.io/Adele/
<br>
Какие задачи перед собой ставил:
<br>
1.Использование при верстке PUG и SCSS.
<br>
1.Использование только нативного JS.
<br>
Что было реализовано:
<br>
1.Верстка, кроссбраузерная, адаптивная, с использование Flex-box, Grid.
<br>
2."Липкий" хедер с гамбургер-меню.<br>
3.При скроле страницы хедер получает прозрачность, уменьшается в высоте.<br>
4.При клике на бургер появляется полупрозрачное меню, замораживается скролл страницы.<br>
5.При клике на пункт меню происходит плавный скролл к соответсвующей секции, с учетом высоты "липкого" хедера.<br>
6.При клике на логотип идет скролл на первую секцию.<br>
7.В секции Real Man реализван анимированный, адаптивный слайдер.<br>
8.Секция TESTIMONIALS реализован слайдер с автоматической прокруткой, которая останавливается при наведении курсора на блок с текстом.<br>
Если убрать курсор - идет переход на следующий слайд.<br>
9.Секция Supplement реализованы табы, при клике на список ингредиентв, меняется блок с описанием.<br>
